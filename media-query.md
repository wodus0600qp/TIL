# 반응형&레이아웃

---

## 반응형이란

---

반응형 웹이란 다양한 화면 크기(모바일, 태블릿, PC)에 맞게 웹 페이지의 레이아웃과 요소들이 자동으로 변경되도록 만드는 것이며, 이를 위해 CSS의 @media를 사용하여 특정 조건에 따라 스타일을 다르게 적용

### 반응형

1.@media → 조건 걸기

2.max-width → 이하일 때 적용 (모바일)

3.min-wi형th → 이상일 때 적용 (PC)

### 반응형 단위

1.% → 부모 기준

2.vw → 화면 가로 기준

3.vh → 화면 세로 기준

4.rem → 기본 글자 크기 기준

---

### FLEX

Flex는 주로 메뉴 정렬이나 버튼 배치처럼 한 줄로 정렬할 때 사용

1.display: flex : 해당 요소를 Flex 컨테이너로 만들어 내부 요소들을 유연하게 배치

2.justify-content : 주축방향에서 요소들의 정렬 방식을 설정하는 속성으로, center, space-between 등을 사용하여 가로 정렬을 조절가능

3.align-items : 교차축 방향에서 요소들의 정렬을 설정하는 속성으로, 세로 정렬을 제어할때사용

---

### GRID

Grid는 전체 페이지 레이아웃처럼 복잡한 배치에 적합

1.display: grid : 해당 요소를 Grid 컨테이너로 만들어 격자 형태의 레이아웃을 사용가능하게함

2.grid-template-columns : 열(column)의 개수와 크기를 설정하는 속성으로, 예를 들어 1fr 1fr과 같이 작성하면 동일한 크기의 열만들기 가능

3.grid-template-rows : 행(row)의 개수와 크기를 설정하는 속성으로, 콘텐츠의 높이나 비율에 따라 행을 구성가능

---

## FLEX와 GRID의 차이

-Flex는 한 방향(가로 또는 세로)으로 요소를 정렬하는 1차원 레이아웃 방식.

-Grid는 행과 열을 동시에 사용하는 2차원 레이아웃 방식.

따라서 간단한 정렬에는 Flex를, 복잡한 구조 설계에는 Grid를 사용하는 것이 적절.

---
