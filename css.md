    # CSS
---
## CSS란
CSS (Cascading Style Sheets)는 웹페이지의 색상, 글꼴, 배경, 레이아웃 등을 꾸미는 스타일 언어

---
## CSS 기본문법
CSS는 선택자 + 속성 + 값으로 작성
## SELECTOR 
selector은 css를 적용할 html을 선택하는 방법
1. 태그 선택자 -> 특정 태그 전체에 스타일 적용
2. class 선택자 -> 여러 요소에 같은 스타일 적용
3. id 선택자 -> 하나의 요소에 적용
---
## 기초 스타일링
## 글꼴(font)
글자의 크기, 글꼴, 굵기를 설정
font-size : 글자 크기
font-family : 글꼴
font-weight : 글자 굵기

## 색상(color)
글자의 색 설정

color : 글자 색

## 배경(background)
요소의 배경색이나 배경이미지 설정
background-color : 배경 색

---
## BOX MOLDEL
1.margin : 요소 바깥쪽 여백

2.padding : 내용과 테두리 사이의 안쪽 여백

3.border : 요소의 테두리
