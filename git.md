# HTML
---
## HTML
---
HTML이란 웹페이지를 만드는 기본 언어로, 웹사이의 구조와 내용을 만드는 일을 한다
즉, 웹 페이지의 뼈대 역할

---

## HTML의 기본구조
!DOCTYPE html-> HTML문서 선언

html -> 전체 HTNL영역

head -> 페이지 정보

body -> 화면의 보이는 내용
  
---
  
## 자주 사용하는 HTML태그
---

div -> 웹페이지를 구역으로 나눌때 사용하는 태그

span -> 짧은 텍스트 부분을 나눌때 사용하는 태그

p -> 문단이나 단락을 만들때 사용

a -> 링크를 만들어 다른 페이지로 이동

img -> 이미지를 화면에 표시

form -> 사용자 입력을 받는 폼을 만들때 사용

input -> 텍스트나 비밀번호를 입력하는 입력창

button -> 버튼을 만들때 사용

---

## 시맨틱 태그

---

header -> 페이지나 섹션의 머리부분으로 로고나 제목이 들어감

footer -> 페이지나 섹션의 맨 아래부분, 보통 추가정보나 저작권등이 들어감

main -> 페이지의 핵심내용이 들어가는 부분으로 보통 한페이지에 하나만 사용

section -> 주제별로 내용을 묶는영역으로 하나의 주제를 가진 컨텐츠 묶음
