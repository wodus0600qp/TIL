# GIT
---
## GIT이란
---
1.Repository -> 프로젝트와 코드가 저장되는 공간

2.COMMIT -> 현재상황을 스냅샷처럼 저장하듯 코드 변경상황을 기록

3.BRANCH -> 코드를 나눠서 작업하는 기능

4.Staging Area -> 중간저장공간으로 커밋하기전에 잠깐 올려두는 

---
## GIT명령어
---
1.GIT ADD -> 파일을 저장하기전에 준비하는 단계

2.GIT COMMIT -> 변경사항을 실제로 저장하는 단계

3. GIT PUSH -> 코드를 CITHUB에 업로드하는 단계
