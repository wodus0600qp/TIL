## JAVASCRIPT란

---

JavaScript는 웹 페이지의 동작을 제어하기 위한 프로그래밍 언어로, 변수, 조건문, 반복문, 함수, 자료구조 등의 기본 문법을 사용하여 프로그램을 작성

---

## 기본 문법

---

### 변수

변수 : let,const

-let -> let은 값을 저장하기 위한 변수를 선언할 때 사용하는 키워드로, 선언 이후에도 값을 자유롭게 변경할 수 있으며, 블록 범위를 가지기 때문에 {} 안에서만 유효

-const -> const는 상수를 선언할 때 사용하는 키워드로, 한 번 값을 할당하면 재할당이 불가능하며, 마찬가지로 블록범위를 가지고 주로 변하지 않는 값을 저장할 때 사용

---

조건문 : if,else

-if -> if문은 주어진 조건이 참일 때 특정 코드를 실행하는 조건문이며, 조건식의 결과에 따라 코드의 실행 여부가 결정

-else -> else는 if문의 조건이 거짓일 때 실행되는 부분으로, 조건에 맞지 않는 경우의 처리를 담당

---

### 반복문

반복문 : for,while

-for -> for문은 초기값, 조건식, 증감식을 이용하여 반복 횟수가 정해진 경우에 사용하는 반복문이며, 정해진 횟수만큼 코드를 반복 실행할 때 적합

-while -> while문은 조건식이 참인 동안 계속해서 코드를 반복 실행하는 반복문이며, 반복 횟수가 정해져 있지 않을 때 주로 사용

---

### 함수

함수 : 함수선언,화살표 함수

-함수선언 -> 함수 선언은 function 키워드를 사용하여 특정 기능을 수행하는 코드를 정의하는 방법이며, 필요할 때 호출하여 재사용가능

화살표 함수 -> 화살표 함수는 => 기호를 사용하여 함수를 간단하게 표현하는 방식으로, 코드가 짧고 간결해지는 장점이 있으며, 주로 짧은 함수나 콜백 함수에서 많이 사용

---

### 자료구조

자료구조 : 배열,객체

-배열 -> 배열은 여러 개의 데이터를 순서대로 저장하는 자료구조로, 각 데이터는 인덱스를 통해 접근할 수 있으며, 같은 타입 또는 다른 타입의 데이터를 함께 저장가능

-객체 -> 객체는 키와 값의 쌍으로 데이터를 저장하는 자료구조이며, 각 데이터는 이름(키)을 통해 접근할 수 있어, 의미 있는 데이터 구조를 만들 때 유용
