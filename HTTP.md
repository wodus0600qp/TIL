# HTTP 기본개념

---

## HTTP

인터넷에서 데이터를 주고받는 기본규칙으로 암호화는 X, 내용이 그대로 다보임

---

## HTTPS

HTTPS는 HTTP+암호화로 데이터를 암호화해서 전송

## HTTPS가 필요한 이유

1.개인정보 보호

2.해킹이나 도청 방지

3.데이터 위조방지

---

## REQUSET/RESPONSE

### REQUSET

사용자가 서버에게 요청하는것

#### 구성

-`URL`

-`METHOD`

-`데이터`

---

### RESPONS

서버가 사용자에게 보내는 답

#### EX)

-웹페이지HTML

-데이터

-오류메세지

#### 흐름

클라이언트->요청->서버->응답->클라이언트

---

## HTTPMETHOD

### GHE

-데이터 조회

-서버에서 가져오기만

EX)게시글 보기

---

### POST

-데이터 생성

-서버에 새로운기데이터 추가

EX)회원가입,글쓰기

---

### PUT

-데이터 수정

-기존 데이터를 통째로 변경

EX)게시글 전체 수정

---

### DELETE

-데이터 삭제

EX)게시글 삭제

---

## 상태 코드

서버가 응답할때 숫자로 알려줌

- `200` (성공)
- `201` (생성 성공)
- `400` (잘못된 요청)
- `401`, `403` (인증 / 권한)
- `404` (찾을 수 없음)
- `500` (서버 오류)
