# 이벤트 처리

---

## 이벤트처리란

---

이벤트 처리는 사용자의 행동(클릭, 입력 등)에 반응하여 특정 동작을 실행하는 것을 의미하며, JavaScript에서는 이벤트 리스너를 통해 이벤트가 발생했을 때 실행할 함수를 등록가능

---

## 이벤트리스너

---

이벤트 리스너 : addEventListener

-addEventListener -> 특정 요소에 이벤트를 등록하는 메서드이며,어떤 이벤트가 발생했을 때 어떤 함수를 실행할지 설정할 수 있으고 여러 개의 이벤트를 동시에 등록 가능

---

## 주요 이벤트

---

주요 이벤트 : click,input,change

-click -> 사용자가 요소를 클릭했을 때 발생하는 이벤트이며, 버튼 클릭 처리 등에 주로 사용

-input -> 사용자가 입력창에 값을 입력할 때마다 발생하는 이벤트이며, 실시간으로 값을 처리할 때 사용

-change -> 입력값이 변경되고, 포커스를 잃었을 때 발생하는 이벤트이며, 입력이 완료된 후에 실행

---

## 이벤트 객체

---

이벤트 객체 : event, event.target

-event -> 이벤트가 발생했을 때 자동으로 생성되는 객체로, 이벤트에 대한 다양한 정보를 담고 있음

-event.target -> 이벤트가 실제로 발생한 요소를 가리키는 속성이다. 어떤 요소에서 이벤트가 발생했는지 확인할 때 사용
