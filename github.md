# github
---
## github란
--- 

GitHub는 Git을 사용하는 온라인 코드 저장소 플랫폼으로,
개발자가 만든 프로그램을 저장하고 공유하며 협업할 수 있는 서비스

---
## github의 주요기능
---
1. 원격 저장소 -> 프로젝트를 인터넷에 저장하는 공간으로 언제 어디서든 확인 가능
2. 협업기능 -> 여러사람이 같은 프로젝트가 가능하며, 공유가능
3. 버전관리 -> 코드변경기록이 자동으로 저장되며, 이전 버전으로 되돌리기 가능
4. 브랜치 -> 내가 수정한 코드를 다른 사람에게 검토요청이 가능하며 팀 프로젝트에 많이 사용
